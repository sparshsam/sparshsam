# Security Policy

This repository is the public profile and research index for the Sparsh Sam open-source ecosystem. It does not run production services or store application secrets.

## Reporting A Vulnerability

Please report security issues through GitHub by opening a private vulnerability report where available, or by contacting the maintainer directly through the profile-linked channels.

Do not publish exploit details, credentials, private keys, personal data, or sensitive operational information in public issues.

## Scope

In scope:

- Broken links or misleading security claims in profile documentation.
- Accidental disclosure of sensitive implementation details.
- Security issues in shared templates that could affect downstream repositories.

Out of scope:

- General feature requests.
- Vulnerabilities in third-party services not controlled by this ecosystem.
- Requests for private repository access.

For project-specific issues, use the `SECURITY.md` file in the affected repository.
