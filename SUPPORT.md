# Support

This profile repository is a public index for research, open-source projects, and ecosystem standards.

## Where To Ask

- For a specific project, open an issue in that project's repository.
- For ecosystem documentation or standards, use [ecosystem-standards](https://github.com/sparshsam/ecosystem-standards).
- For security concerns, follow [SECURITY.md](SECURITY.md).

## Maintainer Expectations

This is an independent research and open-source ecosystem. Response times may vary, and support is best-effort unless a project states otherwise.
