# Contributing

This profile repository is maintained as a public research and ecosystem index. Contributions are welcome when they improve accuracy, clarity, documentation quality, or shared templates.

## Useful Contributions

- Fix broken links, typos, or stale project references.
- Improve shared templates for repository metadata, security, citation, or contribution workflows.
- Suggest clearer language for describing public/private boundaries.
- Propose documentation updates that make the ecosystem easier to inspect.

## Process

1. Open an issue or pull request with a focused change.
2. Keep language restrained, factual, and specific.
3. Avoid adding private implementation details or unverified claims.
4. Follow the repository standards in [ecosystem-standards](https://github.com/sparshsam/ecosystem-standards).
