# Recommended Future Projects

These are missing pieces that would complete the ecosystem naturally without expanding into unrelated product territory.

## CalmKit

A shared component and interaction library for emotionally restrained UX.

Purpose:

- reusable interface patterns;
- calm color tokens;
- warning and confirmation patterns;
- no-streak, no-shame progress displays;
- accessibility-first controls.

## ProofNotes

Local-first signed research notes with optional public timestamp proofs.

Purpose:

- bridge OpenProof and the research identity;
- support citeable lab notes;
- preserve authorship and revision history.

## LocalArchive

A small tool for exporting, validating, and packaging personal data archives.

Purpose:

- shared export conventions;
- checksums and manifests;
- long-term custody of records.

## CalmRFC

A repository for RFCs, specifications, and shared schemas.

Purpose:

- centralize protocol work;
- assign stable RFC numbers;
- prepare documents for DOI publication.

## CustodyBench

A test suite and checklist for evaluating whether software meaningfully supports user custody.

Purpose:

- assess exportability, reversibility, local operation, auditability, and deletion.
